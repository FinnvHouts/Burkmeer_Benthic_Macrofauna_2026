# Burkmeer_Benthic_Macrofauna_2026
Data from: Paludiculture: Restoring Peatland Biodiversity Through Wet Agriculture? — Bachelor's Thesis, Biology, 2026. Benthic macro-invertebrate abundance, taxon composition, functional feeding groups &amp; Shannon diversity from three peatland sites (drained, paludiculture, restored) in the Burkmeer polder, NL.
